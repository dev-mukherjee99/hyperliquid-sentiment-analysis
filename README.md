# Hyperliquid Sentiment Analysis

## Objective
This project analyzes how Bitcoin market sentiment (Fear/Greed) affects trader behavior and performance on Hyperliquid.

## Files
- Hyperliquid_Sentiment_Analysis.ipynb

## How to Run
1. Download all files
2. Open the notebook in Jupyter or VS Code
3. Run all cells from top to bottom

## Libraries Required
- pandas
- numpy
- matplotlib
- scipy
- scikit-learn

Install using:
pip install pandas numpy matplotlib scipy scikit-learn

## Output
- Charts comparing sentiment vs trader behavior
- Performance analysis
- Segmentation insights
- Strategy recommendations
