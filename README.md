# Hyperliquid Sentiment Analysis

## Objective
This project analyzes how Bitcoin market sentiment (Fear/Greed) relates to trader behavior and performance on Hyperliquid.

## Files Used
- fear_greed_index.csv
- historical_data (1).csv
- Hyperliquid_Sentiment_Analysis.ipynb

## Tasks Covered
- Data loading and audit
- Data cleaning and date alignment
- Daily trader-level metric creation
- Sentiment-based performance analysis
- Sentiment-based behavior analysis
- Trader segmentation
- Actionable strategy recommendations
- Optional predictive modeling

## Libraries Required
- pandas
- numpy
- matplotlib
- scipy
- scikit-learn

## How to Run
1. Keep the notebook and both CSV files in the same folder.
2. Open the notebook in Jupyter Notebook or VS Code.
3. Run all cells from top to bottom.

## Output
The notebook generates:
- summary tables
- charts
- trader segment analysis
- strategy recommendations
- optional model results
